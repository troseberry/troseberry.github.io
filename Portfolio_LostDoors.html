LOST DOORS

Project Overview

Project Duration:

12 Weeks

Team Size:

4

Engine:

Corona SDK

Language Used:

Lua

Role(s):

Programmer

Lost Doors is a maze-based puzzle game where you must guide the adventurer, Sally, through a set of ever-changing mazes.

Each maze has a number of colored doors. Pass through open doors to reach the end portal. Different doors change the mazes in diferent ways. Observe their behavior and find your escape route!

Roles & Responsibilities

Programmer

  • Programmed gameplay systems in Lua
  • Built levels in Corona SDK based on pencil and paper designs
  • Designed and implemented sound effects
  • Provided original game idea and collaborated with group to expand it into a full game
