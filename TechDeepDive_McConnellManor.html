MCCONNELL MANOR - TECHNICAL DEEP DIVES

The Memento Puzzle

Background

The secondary quest the player must complete during the "McConnell Manor" quest is a quest called "Ghost, Interrupted". In this quest, the player must assist a ghost named Jedediah by returning three specific items to 3 different dressers.

It is essentially a puzzle where the player must determine which item belongs in which dreser based on different clues they find in the manor. There is only one correct combination and the player must determine it to move forward with the quest.

Initial Implementation

First, I tried having the containers check their inventory when the player talks to Jed. With this implementation, I tried to have references to the three unique containers (dressers) in the condition for the option to tell Jedediah that the player had returned all the items correctly. This check would look at each container, have the container go through its inventory, check if it contained the specific item it was supposed to, make sure all three container had their items, and then Jedediah would tell the player whether they were correct or not.

Working Solution

In the end, it made more sense that each item would hold a reference to the container it was supposed to be in. When the item changed containers, it would check if the new container was the correct one.

To accomplish this, I created a script that attached to the different items.

The user supplies two properties for the script: a global variable reference that tracks if the object is correctly placed and an object reference for the correct destination container.

If it was correct, it would set that global variable to 1 (for true). If it was incorrect, it would set it to 0 (for false). Then in the conditions for the conversation where Jedediah tells the player they’ve correctly placed all the items; it would just check that those three global variables were all 1. Otherwise, it would not play that conversation.
